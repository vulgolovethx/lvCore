# lvCore
🔐 lvCore - Sistema Base de Licenciamento e Autenticação

⚙️ Núcleo essencial para os plugins LV.

 💡 O que é o lvCore?

O **lvCore** é o sistema base utilizado em todos os plugins da linha LV.

Ele é responsável por:

* 🔐 Autenticação de licença
* 🔑 Validação de uso autorizado
* 🧠 Proteção contra uso não licenciado

 ⚙️ Como funciona

O sistema segue o fluxo: (mesmo fluxo para todos plugins LV, o que muda é que o lvCore é o 1 batalhão de segurança dos plugins LV)

1. O plugin gera um **Server-ID único**
2. O dono envia esse server-ID gerado para mim
3. Eu gero uma **Key de licença**
4. Te mando a licensa e o lvCore/lvMinas/lvMinasPV valida essa Key localmente
5. O plugin é liberado para uso

 🧱 Integração

O lvCore é obrigatório para os seguintes plugins:

* ⛏️ lvMinas
* 🏠 lvMinasPV

 🔒 Segurança

✔ Sistema de licença manual
✔ Validação por servidor único
✔ Proteção contra cópia não autorizada
✔ Bloqueio automático sem licença válida


 ⚠️ Importante

Este sistema não é distribuído livremente.

O lvCore é fornecido apenas para clientes que adquirirem os plugins da linha LV.


 🎯 Objetivo

* Garantir segurança dos plugins
* Evitar uso não autorizado
* Proteger o ecossistema LV
* Manter integridade das licenças
